HTML5 Boilerplate template for Visual Studio 2013

To make deployement easier all website content is inside the www folder, while all Visual Studio configuration files reside on the parent folder. A HTML redirect page is included to make IIS automatically redirect you to the www folder.

This solution, and be extension it's templates, come in two forms, the vanilla HTML5 Boilerplate and an HTML5 Boilerplate without javascript for front end demos.
